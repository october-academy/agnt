# 카피라이팅 프레임워크

랜딩페이지 카피 작성을 위한 공식, 구조, CTA 가이드.

## 헤드라인 공식 (15종)

### Outcome-Focused (결과 중심)

**{원하는 결과} without {고통}**
> 유저가 사이트를 어떻게 경험하는지 숫자에 빠지지 않고 이해하세요

**{원하는 결과}를 {시간} 안에**
> 30일 안에 첫 매출을 만드세요

### Problem-Focused (문제 중심)

**Never {불쾌한 사건} again**
> 다시는 고객을 놓치지 마세요

**{문제}에 지쳤다면**
> 만들고 아무도 안 쓰는 제품에 지쳤다면

**Stop {나쁜 행동}. Start {좋은 행동}.**
> 기능 나열 그만. 고객이 원하는 것을 파세요.

### Audience-Focused (타겟 중심)

**{핵심 기능} for {타겟}**
> 1인 개발자를 위한 30일 PMF 검증 시스템

**{타겟}가 {결과}하는 가장 빠른 방법**
> 사이드프로젝트에서 첫 매출을 만드는 가장 빠른 방법

### Differentiation-Focused (차별화 중심)

**The {기존 방식의 반대} way to {결과}**
> 코드 없이 랜딩페이지를 만드는 가장 쉬운 방법

**{기존 대안} 말고, {우리 방식}**
> YouTube 튜토리얼 말고, 실전 스프린트

### Proof-Focused (증거 중심)

**{숫자} {사람}이 {제품}으로 {결과}**
> 127명의 개발자가 Agentic30으로 첫 매출을 만들었습니다

**{숫자}% {개선} in {기간}**
> 14일 만에 전환율 3배 향상

### Question-Focused (질문 중심)

**{ICP}인데 {문제}로 고민이야?**
> 만들 줄은 아는데 뭘 만들어야 팔리는지 모르겠어?

**왜 {놀라운 결과}가 가능한가**
> 왜 30일이면 충분한가

### Specificity-Focused (구체성 중심)

**{구체적 숫자}의 {구체적 방법}**
> 5개의 질문으로 만들 제품을 찾는 법

## 페이지 구조 (내러티브 아크)

StoryBrand "Guide through Transformation" 패턴 기반 11개 섹션:

```
1. Hero — 헤드라인 + 서브헤드 + CTA (3초 안에 "나를 위한 거다")
2. Social Proof Bar — 로고, 숫자, 한 줄 추천 (신뢰 장벽 해소)
3. Problem — ICP가 겪는 문제를 구체적으로 (인터뷰 실제 표현 사용)
4. How It Works — 3단계 이내 (복잡함을 단순화)
5. Key Benefits — 2-3개 (기능이 아니라 변환: 전→후)
6. Testimonial — 실제 인용 (없으면 생략, 가짜 금지)
7. Use Cases / Personas — "이런 사람에게 딱"
8. Comparison — 대안 대비 차이 (표 또는 before/after)
9. Case Study Snippet — 구체적 결과 + 이름/직업
10. FAQ — 반론 처리 (3-5개)
11. Final CTA — 보증 + 행동 요청 (Hero CTA와 동일)
```

**규칙:**
- Hero에 CTA 반드시 포함 (스크롤 없이 보여야 함)
- Social proof는 있는 것만 (없으면 섹션 자체 생략)
- FAQ는 가장 흔한 반론을 질문 형태로 전환

## CTA 공식

**Formula:** [Action Verb] + [What They Get] + [Qualifier]

### 강한 CTA (사용)

- "무료 체험 시작하기"
- "30일 스프린트 참가하기"
- "{제품명} 시작하기"
- "가이드 받기"
- "지금 만들기 시작"

### 약한 CTA (피하기)

- "제출" → 뭘 제출하는지 불명
- "가입하기" → 뭘 얻는지 안 보임
- "더 알아보기" → 행동이 아님
- "시작하기" → 너무 모호
- "여기를 클릭" → 1998년

### CTA 컨텍스트별 선택

| 상황 | 추천 CTA |
|---|---|
| 아직 제품 없음 | "이메일 남기기" (관심 검증) |
| MVP 있음, 무료 | "지금 시작하기" (사용 검증) |
| MVP 있음, 유료 | "무료 체험 시작" 또는 "구매하기" |
| 대기리스트 | "얼리버드 등록" |
| 컨설팅/서비스 | "상담 예약하기" |

## JTBD Four Forces (전환 심리)

고객이 전환하려면 4가지 힘의 균형이 맞아야 한다:

```
전환을 밀어주는 힘:
  Push — 현재 상황의 불만 ("혼자 삽질이 지겹다")
  Pull — 새 솔루션의 매력 ("30일 안에 첫 매출")

전환을 막는 힘:
  Habit — 기존 방식의 관성 ("YouTube로도 어떻게든 되긴 해")
  Anxiety — 새 솔루션의 불안 ("99,000원이 아까우면?")
```

카피에서 해야 할 일:
1. Push 강화 — 문제 섹션에서 현재 고통을 구체화
2. Pull 강화 — 결과/변환 섹션에서 성공 비전 제시
3. Habit 깨기 — 비교 섹션에서 기존 방식의 한계 지적
4. Anxiety 해소 — FAQ와 보증에서 위험 제거
