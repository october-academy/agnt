# Seven Sweeps — 카피 품질 체크

7개 차원으로 카피를 순차 점검. 각 sweep에서 이전 sweep도 재확인하여 재귀적 품질 향상.

## Sweep 1: Clarity (명확성)

- [ ] 초등학생이 읽어도 무슨 제품인지 알 수 있는가?
- [ ] 전문 용어를 ICP가 모르는 단어로 쓰지 않았는가?
- [ ] 한 문장에 하나의 아이디어만 담았는가?
- [ ] 수동태 없이 능동태로 썼는가?

## Sweep 2: Voice & Tone (목소리)

- [ ] 처음부터 끝까지 톤이 일관적인가? (반말↔존칭 혼재 없음)
- [ ] 브랜드 성격에 맞는가? (기술적이되 실행 중심)
- [ ] AI 냄새 나는 단어를 쓰지 않았는가? (anti-ai-slop.md 참조)
- [ ] 실제 대화에서 이렇게 말할 수 있는가?

## Sweep 3: So What (그래서 뭐)

- [ ] 모든 문장이 "왜 ICP가 신경 써야 하는지" 답하는가?
- [ ] 기능 나열이 아니라 결과/변환을 말하는가?
- [ ] "그래서 뭐?"라고 물었을 때 대답이 바로 나오는가?
- [ ] 제거해도 의미가 같은 문장이 있는가? → 제거

## Sweep 4: Prove It (증거)

- [ ] 모든 주장에 근거가 있는가?
- [ ] 숫자가 구체적인가? ("많은" → "127명")
- [ ] 후기/인용이 있다면 실제인가? (가짜 금지)
- [ ] 증거가 없는 주장은 "~할 수 있다" 대신 행동 유도로 전환

## Sweep 5: Specificity (구체성)

- [ ] 추상적 약속 대신 구체적 결과를 말하는가?
- [ ] 대상이 구체적인가? ("사람들" → "퇴사 후 전업한 1인 개발자")
- [ ] 시간 프레임이 있는가? ("빠르게" → "30일 안에")
- [ ] 가격/조건이 명확한가?

## Sweep 6: Heightened Emotion (감정)

- [ ] ICP의 고통을 건드리는 문장이 있는가?
- [ ] 성공 비전이 감정적으로 와닿는가?
- [ ] "이건 나 얘기다"라고 느낄 만한 구체적 상황 묘사가 있는가?
- [ ] 과장하지 않으면서도 감정을 움직이는가?

## Sweep 7: Zero Risk (위험 제거)

- [ ] 행동하지 않을 이유를 모두 제거했는가?
- [ ] 보증/환불 정책이 명시되어 있는가? (있다면)
- [ ] 가격 대비 가치가 명확한가?
- [ ] CTA 직전에 마지막 불안을 해소하는 문장이 있는가?
- [ ] 행동의 첫 단계가 쉽게 느껴지는가? ("5분이면 시작")
