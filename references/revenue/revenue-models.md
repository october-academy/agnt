# 수익 모델 가이드

1인 개발자를 위한 8가지 수익 모델. 상황별 추천, 가격대, 첫 매출 경로.

## 모델 한 눈 비교

| # | 모델 | 예상 가격대 | 첫 매출까지 | 필요 트래픽 | 필요 청중 | 반복 매출 | 난이도 |
|---|------|------------|------------|------------|-----------|-----------|--------|
| 1 | 1:1 컨설팅 | 5-30만원/회 | 1-2주 | 최소 | 최소 | ❌ | 하 |

| 4 | 1:N 수업/코호트 | 10-50만원 | 2-4주 | 소-중 | 중 | ❌ | 중 |

## 상세 가이드

### 1. 1:1 컨설팅 / 코칭

**적합 조건:**
- 특정 분야 전문성이 있다 (개발, 디자인, 마케팅, 창업 등)
- 시간을 직접 투입할 수 있다 (주 5-10시간)
- 아직 제품이 없거나 검증 중이다

**가격대:** 시간당 5-30만원 (경력/분야에 따라)
- 주니어: 시간당 5-10만원
- 시니어: 시간당 15-30만원
- 패키지: 4회 묶음 20-80만원

**첫 매출 경로:**
1. 전문 분야 콘텐츠 3-5개 게시 (Threads, LinkedIn)
2. "무료 30분 상담" CTA로 리드 확보
3. 무료 상담 → 유료 전환 (전환율 20-30%)

**장점:** 즉시 시작 가능. 제품 없어도 됨. 고객 이해도 높아짐.
**단점:** 시간 = 매출. 스케일 불가. 번아웃 위험.

### 2. 전자책 / 디지털 템플릿

**적합 조건:**
- 특정 주제에 대해 체계적으로 정리할 수 있다
- 글쓰기에 거부감이 없다
- 트래픽이 있는 채널이 1개 이상 있다

**가격대:** 9,900-49,000원
- 미니 가이드 (20-30p): 9,900-19,900원
- 실전 가이드 (50-100p): 29,000-49,000원
- 템플릿 번들: 19,900-39,000원

**첫 매출 경로:**
1. 블로그/SNS에서 문제 관련 콘텐츠 게시
2. 리드 마그넷 (무료 챕터/미리보기) → 이메일 수집
3. 유료 풀버전 판매
4. 판매 채널: 래피드, Gumroad, 자체 사이트

**장점:** 만들면 반복 판매. 재고 없음. 수정 용이.
**단점:** 마케팅 필요. 경쟁 심함. 평균 구매율 낮음.

### 3. SaaS 구독

**적합 조건:**
- 반복 사용 가치가 있는 소프트웨어를 만들 수 있다
- 지속적인 유지보수/개선이 가능하다
- 최소 월 100명의 방문자 유입이 있다

**가격대:** 월 4,900-49,000원
- 개인용: 월 4,900-9,900원
- 프로/팀: 월 19,900-49,000원
- Freemium + Pro: 무료 → 월 9,900원

**첫 매출 경로:**
1. MVP 배포 + 무료 티어 제공
2. 피드백 수집 → 유료 기능 결정
3. 프리미엄 전환 유도 (기능 제한 또는 사용량 제한)
4. 결제: TossPayments (사업자), Lemon Squeezy (MoR)

**장점:** 반복 매출(MRR). 스케일 가능. 높은 밸류에이션.
**단점:** 개발 시간 많음. 이탈 관리 필요. 첫 매출까지 오래 걸림.

### 4. 1:N 수업 / 코호트

**적합 조건:**
- 가르칠 수 있는 전문 지식이 있다
- 5-20명 단위 그룹 운영이 가능하다
- 이메일 리스트 또는 SNS 팔로워가 100명 이상

**가격대:** 99,000-499,000원
- 미니 워크숍 (2시간): 99,000원
- 코호트 (4주): 199,000-399,000원
- 부트캠프 (8주): 399,000-999,000원

**첫 매출 경로:**
1. 무료 웨비나/라이브로 관심 확인
2. 5명 한정 파일럿 코호트 (할인가)
3. 후기 확보 → 정가 코호트 모집
4. 결제: 래피드, 자체 결제

**장점:** 높은 단가. 커뮤니티 형성. 입소문 효과.
**단점:** 시간 고정. 모집 매번 필요. 최소 인원 리스크.

### 5. 광고 수익

**적합 조건:**
- 월 10,000+ 방문자가 있는 웹사이트/앱이 있다
- 콘텐츠가 지속적으로 쌓이고 있다
- 직접 판매할 제품이 아직 없다

**가격대:** CPM $1-10 (1,000회 노출당)
- 블로그: 월 10만 PV → 월 10-50만원
- 앱: DAU 1,000 → 월 30-100만원

**첫 매출 경로:**
1. 콘텐츠 확보 (블로그 30+개, 또는 앱 DAU 500+)
2. Google AdSense 또는 AdMob 연동
3. 트래픽 증가에 따라 수익 비례 증가

**장점:** 수동적 수익. 제품 없어도 가능. 트래픽 비례.
**단점:** 많은 트래픽 필요. 단가 낮음. 사용자 경험 저하 가능.

### 6. 스폰서 / 후원

**적합 조건:**
- 특정 니치의 청중(뉴스레터, 유튜브, 블로그)이 있다
- 구독자/팔로워가 500명 이상
- 주기적인 콘텐츠를 발행하고 있다

**가격대:**
- 뉴스레터: 구독자 1명당 50-200원/회
- 유튜브: CPM $5-20
- GitHub Sponsors: 후원자당 $5-50/월

**첫 매출 경로:**
1. 뉴스레터/채널 구축 (최소 500 구독자)
2. 스폰서 제안서 작성 (도달, 타겟 프로필, 가격)
3. 관련 기업에 직접 제안 또는 스폰서 플랫폼 등록

**장점:** 콘텐츠와 시너지. 정기 수익 가능.
**단점:** 청중 규모 필요. 스폰서 관리 부담. 콘텐츠 독립성 저하 가능.

### 7. 제휴 마케팅 (Affiliate)

**적합 조건:**
- 블로그, 유튜브, SNS 등 콘텐츠 채널이 있다
- 리뷰, 비교, 추천 콘텐츠를 만들 수 있다
- 사업자 등록 없이 시작하고 싶다

**수수료:** 판매가의 1-10%
- 쿠팡 파트너스: 3-3.5% (24시간 쿠키)
- 리더스CPS: 2-8% (도서/교육)
- 텐핑: 1-5% (종합쇼핑)
- Amazon Associates: 1-10% (카테고리별)
- Lemon Squeezy Affiliates: 판매자 기준 추가 3%, 어필리에이트 기준 2% 수수료 구조 확인 필요

**첫 매출 경로:**
1. 본인 제품/콘텐츠와 연관된 제휴 프로그램 가입
2. 리뷰/비교 콘텐츠에 자연스럽게 링크 삽입
3. SEO + SNS로 트래픽 유입
4. 클릭 → 구매 전환 시 수수료 적립

**장점:** 사업자 불필요. 재고/CS 없음. 콘텐츠와 시너지. 수동 수익 가능.
**단점:** 수수료율 낮음. 쿠키 기간 짧음 (쿠팡 24시간). 플랫폼 정책 변경 리스크.

**한국 특이사항:** 쿠팡 파트너스는 사업자 없이 개인으로 가입 가능. 월 정산, 원천징수 3.3%. 매출 규모가 커지면 사업자 등록 고려.

### 8. 디지털 에셋 판매

**적합 조건:**
- Notion 템플릿, Figma 파일, 코드 보일러플레이트, 프롬프트 팩 등 재사용 가능 에셋을 만들 수 있다
- 특정 직군/용도에 특화된 노하우가 있다
- 한번 만들면 수정 없이 반복 판매 가능한 형태

**가격대:** 5,000-99,000원
- Notion 템플릿: 5,000-29,000원
- Figma UI Kit: 19,000-99,000원
- 코드 보일러플레이트/스타터: 29,000-99,000원
- 프롬프트 팩: 9,900-29,000원
- 엑셀/스프레드시트 템플릿: 5,000-19,000원

**첫 매출 경로:**
1. 본인이 실제 사용하는 에셋을 범용화
2. 미리보기/데모 버전 무료 배포 (리드 마그넷)
3. Gumroad, Lemon Squeezy, 래피드에서 판매
4. SNS/커뮤니티에서 사용 사례 공유

**장점:** 한번 만들면 반복 판매. 재고/배송 없음. MoR로 사업자 불필요. 업데이트로 재구매 유도 가능.
**단점:** 경쟁 심함 (특히 Notion 템플릿). 차별화 필수. 마켓플레이스 수수료 (Gumroad 10%, Lemon Squeezy 5%+50센트 기본 + 일부 추가수수료 가능).

**전자책과의 차이:** 전자책은 "읽는" 콘텐츠, 디지털 에셋은 "쓰는" 도구. 에셋은 구매자가 바로 업무에 적용 가능하므로 전환율이 더 높은 경향.

## 추천 조건표

| 조건 | 1순위 | 2순위 | 비추천 |
|------|-------|-------|--------|
| 제품 없음 + 전문성 있음 | 1:1 컨설팅 | 전자책 | SaaS, 광고 |
| 제품 없음 + 사업자 없음 | 제휴 마케팅 | 디지털 에셋 | SaaS, 코호트 |
| 제품 있음 + 트래픽 없음 | 전자책 | 디지털 에셋 | 광고, 스폰서 |
| 제품 있음 + 트래픽 있음 | SaaS 구독 | 제휴 마케팅 | 1:1 컨설팅 |
| 콘텐츠 채널 있음 (500+) | 스폰서 | 제휴 마케팅 | — |
| 시간 적음 (주 5시간 미만) | 디지털 에셋 | 제휴 마케팅 | 1:1, 코호트 |
| 빠른 첫 매출 필요 | 1:1 컨설팅 | 제휴 마케팅 | SaaS, 광고 |
| 사업자 없이 시작 | 제휴 마케팅 | 디지털 에셋 (MoR) | SaaS (국내 PG) |

## 가장 빠른 첫 매출 경로 (우선순위)

1. **1:1 컨설팅** — 제품 없이 1-2주 안에 가능
2. **제휴 마케팅** — 콘텐츠 있으면 2-4주. 사업자 불필요
3. **전자책/디지털 에셋** — 기존 지식/도구 정리로 2-4주
4. **파일럿 코호트** — 5명만 모으면 2-4주
5. **SaaS 프리미엄** — MVP 있으면 4-8주
6. **스폰서** — 청중 있으면 4-8주
7. **광고** — 트래픽 있으면 8-12주
