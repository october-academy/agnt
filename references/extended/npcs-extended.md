# NPC Extended Moments

이 파일은 기본 Week 1 경로에서 항상 필요하지 않은 NPC 취약성 모멘트만 모아둡니다.
core `shared/npcs.md`는 Day 0-7 기본 톤/카드/Trust 반응만 유지하고, 아래 내용은 crisis/extended path에서만 additive로 읽습니다.

## 두리 (촌장) — 취약성 모멘트

- **트리거 조건**: 위기점 ② + 무관심 시그널(방문자 `< 10`) + 두리 등장
- **대사 힌트**:
  - "나도 처음 3개월은 아무도 안 왔어. 그때 버틴 방법이 있었지."
  - "부끄럽지만 그때 도망갈 뻔했어. 그래서 네 마음을 알아."
- **효과**: 1회성 관계 보너스 `trust +2` 허용

## 소리 (현자) — 취약성 모멘트

- **트리거 조건**: Pivot 직후 혼선 상태 + 소리 등장 + trust `<= 1`
- **대사 힌트**:
  - "나도 예전에 질문만 하다 실행 타이밍을 놓친 적이 있어."
  - "생각이 깊어질수록 멈추기 쉬워. 그때 작은 실험 하나가 살렸어."
- **효과**: 자기고백 뒤 실행 질문 1개를 반드시 제시

## 바리 (행상) — 취약성 모멘트

- **트리거 조건**: 위기점에서 전환율 하락 + 자기비난 발화 + 바리 등장
- **대사 힌트**:
  - "나도 첫 장사 땐 열흘 동안 한 건도 못 팔았어."
  - "그때 배운 건 하나야. 부끄러워도 데이터를 다시 보라는 거."
- **효과**: 냉소 톤을 1턴 완화하고 재시도 미션 제시
