# 사업자 등록 의사결정 가이드

한국에서 1인 개발자가 사업자를 등록할지 판단하기 위한 의사결정 트리.

> **면책 고지**: 이 가이드는 일반적인 참고 정보입니다. 개인 상황에 따라 결과가 다를 수 있으며, 구체적인 판단은 세무사 또는 국세청(126)에 문의하세요.

## 의사결정 트리

```
현재 상태?
├── 학생
│   ├── 피부양자 + 매출 없음 → DELAY (사업자 보류)
│   ├── 피부양자 + 매출 있음(소액) → DELAY + MoR 활용
│   ├── 지역가입자 + 매출 있음 → CONSIDER (매출 규모 따라)
│   └── 학자금 대출 수혜 중 → DELAY (소득구간 변경 주의)
│
├── 직장인 (정규직)
│   ├── 겸업금지 조항 있음 → DELAY + MoR 우회
│   ├── 겸업금지 없음 + 매출 없음 → DELAY
│   ├── 겸업금지 없음 + 매출 50만+ → CONSIDER
│   └── 퇴사 예정 → REGISTER (퇴직 후 즉시 가능하도록)
│
├── 프리랜서 / 무직
│   ├── 지역가입자 + 매출 없음 → DELAY
│   ├── 지역가입자 + 매출 50만+ → REGISTER (간이과세자)
│   └── 피부양자 + 매출 없음 → DELAY
│
└── 병역 (산업기능요원/전문연구요원)
    └── 모든 경우 → DELAY (겸직 금지 저촉 가능, 관할 병무청 확인)
```

## 판정 3종

| 판정 | 의미 | 다음 행동 |
|------|------|-----------|
| **REGISTER** | 등록 권장 | 간이과세자로 등록, 통신판매업 신고 |
| **CONSIDER** | 상황 따라 | 세무사 상담 후 결정, 당장은 MoR 가능 |
| **DELAY** | 보류 권장 | MoR 서비스로 첫 매출 검증, 매출 확인 후 재판단 |

## 패널티 목록 (사업자 등록 시 발생할 수 있는 영향)

| # | 항목 | 영향 | 상세 |
|---|------|------|------|
| 1 | **건강보험** | 피부양자 탈락 → 지역가입자 | 최저 월 19,500원 (1년차 80% 감면 가능). 소득 없으면 최저 수준 유지 |
| 2 | **국민연금** | 지역가입자 납부 의무 | 소득 없으면 납부예외 신청 가능. 소득 발생 시 소득의 9% |
| 3 | **가족수당** | 부모님 회사 가족수당 자격 상실 가능 | 회사별 기준 다름 — 부모님께 확인 필요 |
| 4 | **예비창업패키지** | 지원 자격 상실 | 이미 사업자가 있으면 "예비" 창업자 아님 |
| 5 | **학자금 대출** | 소득구간 변경 가능 | 한국장학재단 소득분위 재산정 시 영향 |
| 6 | **병역** | 겸직 금지 저촉 | 산업기능요원/전문연구요원 — 관할 병무청 확인 필수 |
| 7 | **세무 관리** | 기장 의무 발생 | 간편장부 대상자면 직접 가능. 연 매출 7,500만원 초과 시 복식부기 |

## 월 비용 시뮬레이션 (사업자 등록 + 매출 없음 가정)

| 항목 | 월 비용 | 비고 |
|------|---------|------|
| 건강보험 (최저) | ~19,500원 | 1년차 80% 감면 적용 시 ~3,900원 |
| 국민연금 | 0원 | 납부예외 신청 시 |
| 기장 | 0원 | 간편장부 직접 처리 시 |
| 통신판매업 | 0원 | 면허세 4,500원/년 (일부 지자체 면제) |
| **합계** | **~19,500원** | 감면 적용 시 ~3,900원/월 |

## 월 비용 시뮬레이션 (사업자 등록 + 월 매출 100만원 가정)

| 항목 | 월 비용 | 비고 |
|------|---------|------|
| 건강보험 | ~50,000-80,000원 | 소득 기반 산정 |
| 국민연금 | ~90,000원 | 소득의 9% |
| 부가세 (간이) | 면제~소액 | 연 매출 1억 400만원 미만 간이과세자 |
| 기장 | 0원 | 간편장부 직접 또는 세무사 연 50만원 |
| **합계** | **~140,000-170,000원** | |

## MoR (Merchant of Record) 대안

사업자 없이 합법적으로 결제를 받는 방법:

| 서비스 | 수수료 | 정기결제 | 한국 카드 | 해외 카드 | 적합 |
|--------|--------|----------|-----------|-----------|------|
| 래피드 (Latpeed) | ~1.6% | ❌ | ✅ | ❌ | 디지털 콘텐츠, 템플릿 |
| Lemon Squeezy | 5% + $0.50 | ✅ | ❌ | ✅ | SaaS, 라이선스 |
| Polar | 5% | ✅ | ❌ | ✅ | 오픈소스, 개발자 도구 |
| 3.3% 원천징수 | 3.3% | — | — | — | 용역, 컨설팅, 과외 |

상세 비교: `references/tools/no-biz-payment.md`

## 간이과세자 혜택

연 매출 1억 400만원 미만이면 간이과세자로 등록 가능:

- 부가가치세: 면제 또는 업종별 부가가치율 적용 (1.5-4%)
- 세금계산서: 발행 가능 (2024년부터)
- 전환: 매출 증가 시 일반과세자로 자동 전환

## 청년창업 중소기업 세액감면

사업자 등록의 가장 강력한 인센티브. 조건 충족 시 소득세/법인세를 최대 100% 감면.

### 대상 조건

| 조건 | 요건 |
|------|------|
| **나이** | 창업 당시 만 15~34세 (군 복무 기간 최대 6년 차감) |
| **창업** | 해당 업종 최초 창업 (동일 업종 폐업 후 재창업 제외) |
| **업종** | 조세특례제한법 지정 18개 업종 (아래 참조) |
| **규모** | 중소기업 기준 충족 |

### 1인 개발자 해당 업종

| 업종 | 해당 여부 | 비고 |
|------|-----------|------|
| 정보통신업 (소프트웨어 개발) | ✅ | 대부분의 SaaS, 앱, 웹서비스 |
| 전문/과학/기술 서비스업 | ✅ | 컨설팅, 디자인, R&D |
| 통신판매업 | ✅ | 디지털 콘텐츠, 템플릿 판매 |
| 출판업 | ✅ | 전자책 |
| 광고업 | ✅ | 마케팅 에이전시 |
| 교육 서비스업 | ✅ | 코호트, 온라인 강의 |
| 카페/비알코올 음료점 | ❌ | 감면 대상 아님 주의 |

### 감면율 (2026년 기준)

| 지역 | 청년 감면율 | 일반(비청년) 감면율 |
|------|------------|-------------------|
| 비수도권 + 수도권 인구감소지역 | **100%** | 75% |
| 수도권 과밀억제권역 외 | **75%** | 50% |
| 수도권 과밀억제권역 내 (서울 등) | **50%** | — |

### 감면 기간 및 한도

- **기간**: 최초 소득 발생 연도부터 **5년간**
- **한도**: 연간 최대 **5억원** (2025년 귀속분부터)
- **신청**: 종합소득세/법인세 신고 시 세액감면신청서 별도 제출 (자동 적용 아님)

### 1인 개발자 시뮬레이션

만 28세, 서울에서 SaaS 창업, 연 매출 3,000만원 가정:
- 소득세 약 200만원 → **50% 감면 → 100만원만 납부**
- 5년간 적용 시 총 절감: ~500만원

만 28세, 비수도권에서 SaaS 창업, 연 매출 3,000만원 가정:
- 소득세 약 200만원 → **100% 감면 → 0원 납부**
- 5년간 적용 시 총 절감: ~1,000만원

### 의사결정 포인트

사업자 등록의 패널티(건보, 국민연금, 가족수당)를 감수하더라도 **매출이 발생하면** 세액감면의 절감 효과가 패널티 비용을 상회할 수 있다. 특히:

1. **비수도권 청년** → 100% 감면. 사업자 등록의 최대 인센티브
2. **서울 청년** → 50% 감면. 매출 규모에 따라 판단
3. **34세 이후** → 청년 감면 불가. 나이 마감 전 등록이 유리할 수 있음

> 주의: 업종 코드 등록이 정확해야 감면 적용됨. 홈택스에서 업종 확인 필수.

## 사업자 등록 전환 시점

아래 중 하나 해당 시 사업자 등록을 적극 고려:

1. 연 매출 2,400만원 초과 예상
2. 한국 카드 정기결제 서비스 운영 필요
3. 세금계산서 발행 필요 (B2B)
4. 투자 유치 또는 직원 채용 계획
5. MoR 수수료(5%)가 사업자 비용보다 비싸지는 시점
6. **청년 나이 마감 임박** (만 34세 전에 등록해야 5년 감면 적용)
