# 사업자 없이 결제 받는 방법

한국에서 사업자등록 없이 합법적으로 매출을 내는 방법. 검증 단계에서 불필요한 리스크 없이 첫 매출을 만들기 위한 가이드.

## 왜 사업자 없이 시작해야 하나

사업자등록 시 발생하는 부담:

1. **건강보험** 피부양자 탈락 → 지역가입자 전환 (최저 월 19,500원, 1년차 80% 감면 있음)
2. **국민연금** 지역가입자 대상 (소득 없으면 납부예외 신청 가능)
3. **부모님 회사 가족수당** 자격 상실 가능
4. **기장 의무** — 간편장부로 충분하지만 관리 필요
5. **병특 복무 중** 겸직 금지 저촉 가능
6. **학자금 대출** 소득구간 변경 가능
7. **예비창업패키지** 지원 자격 상실

**결론: 매출이 확인된 후에 사업자등록. 먼저 검증.**

## 방법 1: 간편결제 앱 (국내)

### 래피드 (Latpeed)

| 항목 | 내용 |
|------|------|
| 수수료 | Free 1.6%, Lite 4.6%, Pro 1.6% (+ PG 별도) |
| 한도 | 플랜/인증 상태에 따라 달라짐 |
| 사업자 | 불필요 |
| 정기결제 | Pro에서 멤버십 지원 |
| 해외 카드 | 미지원 |
| 적합 | 디지털 콘텐츠, 템플릿, 가이드, PDF |

가입 즉시 판매 가능. 크리에이터 특화 플랫폼.

### PayApp

| 항목 | 내용 |
|------|------|
| 수수료 | ~4% |
| 한도 | 연 2,400만원 |
| 사업자 | 불필요 |
| 정기결제 | 미지원 |
| 적합 | 소규모 거래, 용역, 과외 |

수수료가 높지만 간편. 링크 하나로 결제 가능.

### 딜앱 (DealApp)

| 항목 | 내용 |
|------|------|
| 수수료 | 3.5~4% |
| 한도 | 연 5,000만원 |
| 사업자 | 불필요 |
| 적합 | 소규모 거래, 중고거래 |

## 방법 2: MoR 서비스 (해외 결제)

MoR(Merchant of Record)은 판매자 대신 결제를 처리하고 세금 신고까지 대행한다. 한국 사업자 없이 글로벌 매출 가능.

### Lemon Squeezy

| 항목 | 내용 |
|------|------|
| 수수료 | 5% + $0.50/건 기본, 국제결제/PayPal/구독은 추가 수수료 가능 |
| 사업자 | 불필요 |
| 정기결제 | 지원 |
| VAT 처리 | 자동 |
| 적합 | SaaS, 디지털 상품, 라이선스 |

체크아웃 페이지 자동 생성. Stripe 대안으로 1인 개발자에게 인기.

### Polar

| 항목 | 내용 |
|------|------|
| 수수료 | 4% + $0.40/건 기본, 국제카드/구독 추가 수수료 가능 |
| 사업자 | 불필요 |
| 정기결제 | 지원 |
| 적합 | 오픈소스, 개발자 도구, CLI |

GitHub 통합. 오픈소스 프로젝트 수익화에 최적.

### Gumroad

| 항목 | 내용 |
|------|------|
| 수수료 | 10% |
| 사업자 | 불필요 |
| 정기결제 | 지원 |
| 적합 | 전자책, 강의, 디지털 에셋 |

수수료 높지만 마켓플레이스 노출 효과 있음.

## 방법 3: 광고 수익

| 플랫폼 | 사업자 필요 | 비고 |
|--------|------------|------|
| AdMob | 불필요 | 앱 내 광고 |
| AdSense | 불필요 | 웹사이트 광고 |
| 유튜브 파트너 | 불필요 | 영상 광고 수익 |

소규모 수익은 사업자 없이 가능. 지속적·반복적 광고 수익은 사업자등록 의무 발생 가능 (국세청 기준).

## 방법 4: 프리랜서 용역

3.3% 원천징수로 사업자 없이 합법적으로 대금 수취 가능. 클라이언트가 원천징수 후 지급.

- **적합**: 1:1 컨설팅, 코딩 과외, 디자인 용역
- **한도**: 제한 없음
- **주의**: 지속적·반복적이면 사업자등록 의무 발생 가능

## 결정 가이드

```
뭘 팔아?
├── 디지털 콘텐츠 (PDF, 템플릿, 가이드)
│   ├── 한국 고객 → 래피드 (1.6%)
│   └── 글로벌 고객 → Lemon Squeezy (5% + $0.50)
├── SaaS / 구독 서비스
│   ├── 한국 고객 → 사업자등록 후 TossPayments
│   └── 글로벌 고객 → Lemon Squeezy 또는 Polar
├── 용역 / 컨설팅
│   └── 3.3% 원천징수
└── 앱 / 웹사이트 → 광고 수익 (AdMob, AdSense)
```

## 사업자등록 전환 시점

아래 조건 중 하나라도 해당되면 사업자등록을 고려:

1. 연 매출 2,400만원 초과 예상
2. 정기결제 서비스 운영 필요 (한국 카드)
3. 세금계산서 발행 필요 (B2B 거래)
4. 투자 유치 계획
5. 직원 채용 계획

**간이과세자**: 연 매출 1억 400만원 미만이면 간이과세자로 등록 가능. 부가세 면제 또는 경감.
