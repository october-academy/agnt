# 결제 솔루션 비교

1인 개발자/인디해커를 위한 결제 솔루션 비교. 사업자 유무, 예산, 타겟 시장 기준.

|--------|------------|--------|-----------|-----------|-----------|----------|-------------|

| Polar | X | 5% | 0원 | ❌ | ✅ | ✅ | 하 |

## 상세 비교

### TossPayments

**적합**: 한국 시장 타겟, 사업자등록 완료, 월 정기결제 서비스

- **수수료**: 카드 3.3%, 계좌이체 1.5%, 가상계좌 건당 300원
- **정산**: 영업일 기준 D+2 (카드), D+1 (계좌이체)
- **장점**: 한국 최대 PG. SDK/API 문서 우수. 간편결제(토스페이, 네이버페이, 카카오페이) 통합
- **단점**: 사업자등록 필수. 심사 1-3영업일. 테스트 환경 설정 번거로움
- **정기결제**: 빌링키 발급 → 자동 결제. `BillingKey` API
- **연동**: REST API + JavaScript SDK. Next.js/React 예제 공식 제공
- **추천 시나리오**: 한국 유저 대상 SaaS, 온라인 강의, 구독 서비스

### 포트원 (PortOne, 구 아임포트)

**적합**: 여러 PG사를 비교/전환하고 싶을 때, 사업자등록 완료

- **수수료**: 포트원 자체 수수료 없음 (PG사 수수료만 부과)
- **장점**: PG사 추상화 — 토스/이니시스/KCP를 동일 API로. PG 전환 시 코드 변경 최소
- **단점**: 추상화 레이어 추가 = 디버깅 복잡. 포트원 장애 시 전체 결제 불가
- **정기결제**: 지원 (PG사별 차이 있음)
- **추천 시나리오**: 여러 PG사를 테스트하고 싶거나 향후 PG 전환 계획이 있을 때

### 래피드 (Latpeed)

**적합**: 사업자 없이 한국 시장에서 디지털 콘텐츠/템플릿 판매

- **수수료**: ~1.6% (업계 최저 수준)
- **장점**: 사업자등록 불필요. 크리에이터 특화. 가입 즉시 판매 가능
- **단점**: 정기결제 미지원. 해외 카드 미지원. 물리적 상품 부적합
- **한도**: 연 2,400만원 (초과 시 사업자등록 필요)
- **추천 시나리오**: 템플릿, 가이드, 디지털 에셋 판매. 첫 매출 검증용

### Lemon Squeezy

**적합**: 글로벌 시장 타겟, 사업자 없이 SaaS/디지털 상품 판매

- **수수료**: 5% + $0.50/건
- **장점**: MoR(Merchant of Record) — 세금 신고, VAT 처리 대행. 사업자 불필요. 정기결제 기본 제공. Checkout 페이지 자동 생성
- **단점**: 한국 카드 결제 미지원 (해외 카드만). 수수료 높음. 정산 USD
- **정기결제**: 기본 내장. 가격표 + 체크아웃 자동 생성
- **추천 시나리오**: 글로벌 SaaS, 오픈소스 후원, 디지털 상품. 한국 시장만 타겟이면 부적합

### Polar

**적합**: 오픈소스/개발자 도구 수익화, 글로벌 타겟

- **수수료**: 5%
- **장점**: 오픈소스 생태계 통합 (GitHub Sponsors 대안). MoR 서비스. 사업자 불필요
- **단점**: 한국 카드 미지원. 개발자 도구/오픈소스 외 카테고리는 부적합
- **추천 시나리오**: CLI 도구, 라이브러리, 개발자 유틸리티 수익화

## 결정 가이드

```
사업자가 있나?
├── YES → 한국 시장 타겟?
│   ├── YES → TossPayments (직접) 또는 포트원 (추상화)
│   └── NO → Lemon Squeezy 또는 Polar
└── NO → 한국 시장 타겟?
    ├── YES → 래피드 (디지털 콘텐츠, 연 2,400만원 이내)
    └── NO → Lemon Squeezy (SaaS/상품) 또는 Polar (개발자 도구)
```

## 1인 개발자 추천

**첫 매출 검증 단계**: 래피드 (사업자 불필요, 즉시 시작, 수수료 최저)
**검증 후 본격 운영**: TossPayments (한국) 또는 Lemon Squeezy (글로벌)
**PG사 비교/전환 계획**: 포트원
