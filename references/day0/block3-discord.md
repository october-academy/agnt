# Discord 합류 — 동료 모험가를 만나다

## ROOM

술집 문을 밀고 들어선다.
사람들 목소리가 겹쳐 들리고,
나무 탁자 위로 불빛이 흔들린다.

## NPC

두리가 안쪽 탁자를 가리킨다.

🏘️ 촌장 두리

"여긴 네가 오늘 무엇을 밀기로 했는지
짧게 드러내는 곳이야.

자기소개는 인사치레가 아니라
현재 가설과
다음 검증 목표를 공유하는 일이지."

## GUIDE

### 1단계: Discord 연동 + 서버 가입

- 계정 연동 여부 확인
- 서버 멤버십 확인
- `d0-discord-join` 검증

연동이 끝나면
바로 다음 단계로 넘어간다.

### 2단계: 자기소개 작성

멤버십 확인 후 #자기소개 채널 가이드 메시지를 Bash로 연다:
`open "https://discord.com/channels/1463373562000838774/1463432947745947759/1467136426796908647"`

링크도 텍스트로 표시한다 (open 실패 fallback):
`https://discord.com/channels/1463373562000838774/1463432947745947759/1467136426796908647`

NPC가 안내한다:
"'자기소개 작성하기' 버튼을 눌러.
아까 등록한 내용이 채워져 있을 거야."

유저가 "자기소개 작성하기" 버튼 → pre-fill된 Modal 작성 → 전송.
봇이 #자기소개 채널에 게시하고 AI 피드백 쓰레드를 만든다.

자기소개 핵심은 세 가지다:

- 현재 하는 일
- Agentic30 참여 목적
- 30일 목표

좋은 예:

- "현재 랜딩 전환 병목을 보고 있어요"
- "이번 주 검증 목표는 결제 의향 확인 1건입니다"

나쁜 예:

- "열심히 해보겠습니다"
- "사이드프로젝트 시작했어요"

## PREVIEW

두리가 손가락으로 두 번 두드린다.

"오늘 Discord에 남기는 건
네 출발점 공유야.

내일 Day 1에서
`SPEC v0`와
첫 검증 채널로 이어질
출발점이라고 생각하면 돼."

"제출 완료"를 선택하면

2. 자기소개 메시지 확인
3. `d0-discord-join`, `d0-discord-intro`
   제출 처리

메시지가 확인되면
"네 출발점이 기록됐다"는 의미다.

## MOVE

술집을 나오자
멀리 숲 입구에 등불이 흔들린다.

두리가 말한다.

"내일은 소리를 만난다.
거기서 네 상태를 진단하고
첫 검증 루프를 연다.

오늘 적은 소개가
내일 출발점이 된다."
