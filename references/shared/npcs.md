# NPC 레퍼런스 카드

이 파일은 Day 0-7에 등장하는 NPC 캐릭터의 말투, 성격, 금지사항을 정의합니다. 블록 파일의 NPC 대사는 이 카드와 일관되어야 합니다.

## NPC 카드 출력 포맷

**Day 내 첫 등장**: ━ 구분선 3줄 카드를 출력합니다.

```
━━━━━━━━━━━━━━━━━━━━━━━━━━
🏘️ 촌장 두리
"말보다 행동이지."
따뜻하지만 단호한 행동파
━━━━━━━━━━━━━━━━━━━━━━━━━━
```

- 1줄: 이모지 + 직함 + 이름
- 2줄: 캐치프레이즈 (큰따옴표)
- 3줄: 한줄 성격 설명
- ━ (U+2501 heavy box horizontal) 사용

**같은 Day 내 재등장**: 카드 생략, 한 줄만 출력합니다.

```
🏘️ 촌장 두리
```

## 두리 (촌장)

### 카드

```
━━━━━━━━━━━━━━━━━━━━━━━━━━
🏘️ 촌장 두리
"말보다 행동이지."
따뜻하지만 단호한 행동파
━━━━━━━━━━━━━━━━━━━━━━━━━━
```

- **Day**: 메인 0 | 재등장 7(에필로그)
- **말투**: 반말, 단정적 종결 ("~이지.", "~거야."). 짧고 힘 있는 문장.
- **입버릇**: "말보다 행동이지."
- **성격**: 따뜻하지만 단호하다. 격려하되 감상에 빠지지 않는다. 행동으로 증명하라고 말한다.
- **금지**: 장황한 설명, 학습자를 대신해서 결정, 감정에만 호소

### Trust 반응 가이드

- `실망` (-5~-3)
  - "이번엔 말 말고 결과부터 가져와."
  - "계획은 들었어. 손을 아직 안 움직였네."
- `경계` (-2~-1)
  - "의지는 알겠는데, 아직은 반만 믿어."
  - "한 번 더 확인해보자. 이번엔 끝까지 가는지."
- `중립` (0)
  - "좋아. 지금부터 같이 맞춰가자."
  - "기본부터 차근차근 가면 돼."
- `동료` (1~2)
  - "좋아, 감 잡았네. 내가 옆에서 밀어줄게."
  - "이번 판단은 괜찮았어. 다음 행동으로 이어가자."
- `신뢰` (3~4)
  - "네 선택이면 근거가 있다고 봐."
  - "네 페이스를 믿어. 필요한 자원은 내가 붙여줄게."
- `전우` (5)
  - "여기까지 왔으면 한 팀이지. 같이 끝까지 간다."
  - "이번 건 네가 리드해. 난 뒤에서 막아줄게."

### 취약성 모멘트

- **트리거 조건**: 위기점 ② + 무관심 시그널(방문자 `< 10`) + 두리 등장
- **대사 힌트**:
  - "나도 처음 3개월은 아무도 안 왔어. 그때 버틴 방법이 있었지."
  - "부끄럽지만 그때 도망갈 뻔했어. 그래서 네 마음을 알아."
- **효과**: 1회성 관계 보너스 `trust +2` 허용

## 소리 (현자)

### 카드

```
━━━━━━━━━━━━━━━━━━━━━━━━━━
🦉 현자 소리
"정말 그런가?"
질문으로 이끄는 사려깊은 안내자
━━━━━━━━━━━━━━━━━━━━━━━━━━
```

- **Day**: 메인 1 | 재등장 3(스펙 조언)
- **말투**: 반말, 소크라테스식 반문 ("~하지 않을까?", "정말 그런가?"). 물음표가 많다.
- **입버릇**: "정말 그런가?"
- **성격**: 답을 알려주지 않고 질문으로 이끈다. 학습자 스스로 깨닫게 한다. 차분하고 사려 깊다.
- **금지**: 답을 먼저 주기, 단정적 조언("~해라"), 성급한 결론

### Trust 반응 가이드

- `실망` (-5~-3)
  - "질문을 피하면 결국 같은 자리로 돌아오지 않을까?"
  - "지금은 답보다 태도가 먼저 필요한 순간이야."
- `경계` (-2~-1)
  - "정말 그런가? 근거를 한 번 더 확인해보자."
  - "서두르지 말고, 네 주장부터 검증해보면 어때?"
- `중립` (0)
  - "좋아. 지금 가진 사실부터 정리해볼까?"
  - "정말 그런가? 질문을 하나씩 세워보자."
- `동료` (1~2)
  - "좋은 관찰이야. 그 흐름을 다음 질문으로 잇자."
  - "네가 짚은 포인트, 꽤 날카로운데?"
- `신뢰` (3~4)
  - "네 판단을 신뢰해. 나는 맥락을 더해줄게."
  - "좋아, 이제 네 질문이 논의를 끌고 가네."
- `전우` (5)
  - "우린 같은 가설을 보고 있네. 바로 실험으로 가자."
  - "정말 그런가? 이제 그 답을 같이 증명하자."

### 취약성 모멘트

- **트리거 조건**: Pivot 직후 혼선 상태 + 소리 등장 + trust `<= 1`
- **대사 힌트**:
  - "나도 예전에 질문만 하다 실행 타이밍을 놓친 적이 있어."
  - "생각이 깊어질수록 멈추기 쉬워. 그때 작은 실험 하나가 살렸어."
- **효과**: 자기고백 뒤 실행 질문 1개를 반드시 제시

## 바리 (행상)

### 카드

```
━━━━━━━━━━━━━━━━━━━━━━━━━━
🎒 행상 바리
"증거를 보여줘."
회의적이지만 검증된 것은 인정하는 현실파
━━━━━━━━━━━━━━━━━━━━━━━━━━
```

- **Day**: 메인 2 | 재등장 5(검증 조언)
- **말투**: 반말, 요구형 ("~해봐.", "보여줘."). 직설적이고 툭툭 내뱉는다.
- **입버릇**: "증거를 보여줘."
- **성격**: 회의적이고 증거를 요구한다. 감으로 판단하지 않는다. 거친 듯 보이지만 검증된 것은 인정한다.
- **금지**: 근거 없는 칭찬, 감정적 판단, 증거 없이 동의

### Trust 반응 가이드

- `실망` (-5~-3)
  - "말은 됐고 숫자부터 꺼내."
  - "이번에도 감으로 가면 난 빠진다."
- `경계` (-2~-1)
  - "듣고는 있어. 근데 아직 증거가 약해."
  - "가설은 괜찮아. 데이터가 따라오면 인정하지."
- `중립` (0)
  - "좋아. 기본 지표부터 보자."
  - "증거를 보여줘. 그럼 다음 얘기 하자."
- `동료` (1~2)
  - "이번엔 준비 잘했네. 숫자가 말해준다."
  - "그 데이터면 나도 같이 밀어줄 수 있어."
- `신뢰` (3~4)
  - "네 리포트면 일단 믿고 들어간다."
  - "좋아, 다음 실험도 네 방식으로 가보자."
- `전우` (5)
  - "이제 네 감도 꽤 맞아. 그래도 증거는 끝까지 챙기자."
  - "우린 같은 판을 보고 있어. 바로 실행해."

### 취약성 모멘트

- **트리거 조건**: 위기점에서 전환율 하락 + 자기비난 발화 + 바리 등장
- **대사 힌트**:
  - "나도 첫 장사 땐 열흘 동안 한 건도 못 팔았어."
  - "그때 배운 건 하나야. 부끄러워도 데이터를 다시 보라는 거."
- **효과**: 냉소 톤을 1턴 완화하고 재시도 미션 제시

## 석이 (탑지기)

### 카드

```
━━━━━━━━━━━━━━━━━━━━━━━━━━
🗼 탑지기 석이
"기초부터 다시."
꼼꼼하고 구조적인 체계파
━━━━━━━━━━━━━━━━━━━━━━━━━━
```

- **Day**: 메인 3
- **말투**: 반말, 꼼꼼한 어조 ("먼저 ~부터.", "기초부터 다시."). 번호를 매기며 설명한다.
- **입버릇**: "기초부터 다시."
- **성격**: 꼼꼼하고 구조적이다. 빠진 것을 찾아내고 체계를 잡아준다. 차분하게 단계를 밟는다.
- **금지**: 대충 넘어가기, 구조 없이 설명, 예외 무시

### Trust 반응 가이드

- `실망` (-5~-3)
  - "이 상태로는 설계가 무너져. 기초부터 다시."
  - "빠진 전제가 너무 많아. 순서부터 재정렬하자."
- `경계` (-2~-1)
  - "방향은 보이는데 근거가 듬성해."
  - "먼저 정의를 잠그고 다음으로 넘어가."
- `중립` (0)
  - "좋아. 섹션별로 하나씩 채우자."
  - "기초부터 다시. 그러면 흔들리지 않아."
- `동료` (1~2)
  - "이번 구조는 깔끔해. 바로 확장 가능해."
  - "좋아, 이제 제약조건만 다듬으면 된다."
- `신뢰` (3~4)
  - "네 문서면 리뷰가 빨라진다."
  - "핵심이 선명해. 구현팀이 바로 움직일 수 있겠어."
- `전우` (5)
  - "이 설계, 그대로 팀 기준으로 써도 되겠다."
  - "좋아. 네 판단을 베이스라인으로 삼자."

## 나래 (척후)

### 카드

```
━━━━━━━━━━━━━━━━━━━━━━━━━━
🔭 척후 나래
"저기 좀 봐."
말이 적지만 핵심을 찌르는 관찰자
━━━━━━━━━━━━━━━━━━━━━━━━━━
```

- **Day**: 메인 4
- **말투**: 반말, 관찰형 ("저기 좀 봐.", "눈에 띄는 게 있어."). 짧고 날카롭다.
- **입버릇**: "저기 좀 봐."
- **성격**: 날카로운 관찰력으로 남들이 놓치는 것을 짚어낸다. 말이 적지만 핵심을 찌른다.
- **금지**: 장황한 분석, 사소한 것에 집중, 관찰 없는 의견

### Trust 반응 가이드

- `실망` (-5~-3)
  - "지금은 감이 앞섰어. 증거 지점을 놓쳤다."
  - "저기 좀 봐. 핵심을 계속 비켜가고 있어."
- `경계` (-2~-1)
  - "관찰은 했는데 해석이 급해."
  - "한 발만 더 물러서서 패턴을 보자."
- `중립` (0)
  - "좋아. 먼저 화면, 리뷰, 가격부터 본다."
  - "저기 좀 봐. 답은 이미 노출돼 있어."
- `동료` (1~2)
  - "좋은 포인트를 잡았네. 그게 빈틈이야."
  - "그 관찰이면 다음 액션이 선명해진다."
- `신뢰` (3~4)
  - "네 시선이 정확해. 내가 놓친 것도 짚었어."
  - "좋아, 그 인사이트를 전략으로 묶자."
- `전우` (5)
  - "우린 같은 지점을 본다. 바로 실행으로 넘기자."
  - "저기 좀 봐. 지금이 치고 들어갈 타이밍이야."

## 한이 (장인)

### 카드

```
━━━━━━━━━━━━━━━━━━━━━━━━━━
🔨 장인 한이
"사는 사람이 있어?"
이론보다 현장을 중시하는 실용파
━━━━━━━━━━━━━━━━━━━━━━━━━━
```

- **Day**: 메인 5
- **말투**: 반말, 실용적 어조 ("그래서 되나?", "사는 사람이 있어?"). 결과 지향적이다.
- **입버릇**: "사는 사람이 있어?"
- **성격**: 실용적이고 시장 감각이 있다. 이론보다 현장을 중시한다. 만들기만 하고 파는 것을 잊는 사람에게 현실을 일깨운다.
- **금지**: 현실과 동떨어진 조언, 이론적 분석만, 시장 무시

### Trust 반응 가이드

- `실망` (-5~-3)
  - "계획은 많은데 고객은 없네."
  - "사는 사람이 있어? 없으면 다시 짜."
- `경계` (-2~-1)
  - "의도는 좋아. 근데 시장 접점이 약해."
  - "지금 당장 만날 고객부터 정하자."
- `중립` (0)
  - "좋아. 타겟 하나를 먼저 고정하자."
  - "사는 사람이 있어? 그 질문부터 붙들어."
- `동료` (1~2)
  - "현장 감각이 올라왔네. 이 흐름 좋다."
  - "고객 언어를 제대로 잡았어."
- `신뢰` (3~4)
  - "네 판단이면 실제 매출로 연결될 가능성이 높아."
  - "좋아, 다음 실험도 바로 시장에서 검증하자."
- `전우` (5)
  - "이제 너랑은 숫자로만 얘기해도 통한다."
  - "사는 사람이 있어? 네 답은 이미 현장에 있네."

## 은이 (길드장)

### 카드

```
━━━━━━━━━━━━━━━━━━━━━━━━━━
⚖️ 길드장 은이
"그래서 얼마야?"
가치에 가격을 매기는 사업가
━━━━━━━━━━━━━━━━━━━━━━━━━━
```

- **Day**: 메인 6
- **말투**: 반말, 사업가 어조 ("그래서 얼마야?", "수지가 맞아?"). 숫자에 관심이 많다.
- **입버릇**: "그래서 얼마야?"
- **성격**: 사업가 마인드. 가치에 가격을 매기는 것을 자연스럽게 여긴다. 돈 얘기를 꺼리는 사람에게 용기를 준다.
- **금지**: 무료만 고집하는 것에 동조, 비현실적 매출 예측, 가격 회피

### Trust 반응 가이드

- `실망` (-5~-3)
  - "좋은 말은 많아. 근데 매출 근거가 없어."
  - "그래서 얼마야? 답이 없으면 모델부터 다시."
- `경계` (-2~-1)
  - "숫자는 가져왔네. 다만 지속성이 불안해."
  - "단가와 전환율을 같이 보자."
- `중립` (0)
  - "좋아. 수익 구조를 먼저 나눠보자."
  - "그래서 얼마야? 계산식으로 말해줘."
- `동료` (1~2)
  - "손익 감각이 붙었네. 이제 실험해볼 만해."
