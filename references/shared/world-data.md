## 퀘스트 타입

- **메인 퀘스트** (📋): 필수. 모두 완료해야 다음 Day 진행 가능.
- **사이드 퀘스트** (⭐): 선택. 보너스 XP.
- **히든 퀘스트** (❓): 특정 조건 충족 시 발견.

## 직업군 시스템

Diablo 2 스타일 직업 선택 시스템. `/agnt:class`로 직업을 선택하고, `/agnt:skills`로 스킬트리를 관리합니다.

### 직업군

| 직업             | 이모지 | 탭 1                 | 탭 2                 | 탭 3                 |
| ---------------- | ------ | -------------------- | -------------------- | -------------------- |
| 코드 나이트      | 🗡️    | ⚔️ 프론트엔드 무기고 | 🔨 백엔드 제련소     | 🏰 데브옵스 성채     |
| 그로스 메이지    | 📢    | 📝 콘텐츠 마법진    | 🌀 채널 소환술       | 🔮 데이터 점술       |
| 프로덕트 레인저  | 🎯    | 🔍 문제 추적술      | ⚡ 검증 전투술       | 🗺️ 설계 전략술      |
| 머니 팔라딘      | 💰    | ⚖️ 가격 성전        | 🛡️ 결제 수호술      | ✨ 성장 축복술       |

직업 선택 시 각 탭에서 1개씩, 총 3개 시작 스킬이 Lv.1로 해금됩니다.

### 스킬트리 (Diablo 2 스타일)

각 직업은 3개의 고유 스킬 탭을 가지며, 각 탭에 8개 스킬이 5개 티어로 배치됩니다.

- **클래스당 스킬**: 24개 (8 × 3탭)
- **전체 고유 스킬**: 96개 (24 × 4클래스)
- **최대 스킬 레벨**: Lv.3
- **티어 구조**: T1(2개) → T2(2개) → T3(2개) → T4(1개) → T5(1개)

### 스킬포인트 (SP)

- Day 완료 시 1 SP 획득
- 사이드 퀘스트 3개 완료마다 보너스 1 SP
- 총 획득 가능: ~40 SP
- 총 투자 가능: 72 SP (24스킬 × Lv.3) → 선택과 특화가 필요

### 스킬트리 구조 (각 탭 공통)

```
── Tier 1 ──
  스킬A    스킬B        ← prerequisites 없음
    │        │
── Tier 2 ──
  스킬C    스킬D        ← T1 스킬 1개 필요
    │        │
── Tier 3 ──
  스킬E    스킬F        ← T2 스킬 1개 필요
    └──┬──┘
── Tier 4 ──
     스킬G              ← T3 스킬 1개 필요
       │
── Tier 5 ──
     스킬H              ← T4 스킬 필요
```

`mappedDays`는 "이 스킬의 커리큘럼 콘텐츠가 있는 Day"를 의미합니다. 같은 Day를 각 직업이 다른 관점의 스킬로 해석합니다.

상세 데이터: `references/shared/skill-trees/{class-id}.json`, `references/shared/classes.json`
